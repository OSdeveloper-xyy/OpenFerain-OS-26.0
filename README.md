# OpenFerain OS 26.0 ReadMe

## 中文README

首先需要下载 *NASM* ，下载链接如下

![](https://cdn.luogu.com.cn/upload/image_hosting/3atkz3og.png)
[NASM Download](https://www.nasm.us/pub/nasm/releasebuilds/)

然后下载 *x86_64-elf-gcc* ，下载链接如下

![](https://cdn.luogu.com.cn/upload/image_hosting/3atkz3og.png)
[x86_64-elf-gcc](https://github.com/lordmilko/i686-elf-tools/releases)

将 **NASM** 和 **GCC** 添加进系统 **PATH**

点击 *make.bat* 进行编译
